## 步骤

### 1. 修复 GitHub Release tag

release tag 现在是 `v1.0.0`，需要改成 `1.0.0`（不带 v）：

```bash
cd "C:\Users\Ma2431\Desktop\@WrokSpace\Pj4-聊天记录气泡渲染插件"
git push --delete origin v1.0.0
```

这会触发 release workflow，生成新的 Release `1.0.0`（带 main.js/manifest.json/styles.css）。

### 2. 通过新系统提交

1. 打开 **https://community.obsidian.md**，用 Obsidian 账号登录
2. 在个人资料中关联你的 GitHub 账号
3. 进入 **Developer Dashboard** → 点 **"New plugin"**
4. 输入仓库地址：`bruce2431/obsidian-chat-bubble-renderer`
5. 同意开发者政策 → 提交
6. 等待自动化审核（几分钟内出结果）

### 3. 审核通过后

- 插件会在 **24小时内** 上架到社区插件市场
- 后续版本更新只需发布新的 GitHub Release 即可，无需再提交审核
